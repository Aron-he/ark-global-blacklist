服务器只需拉取以下文件：

- github  
  https://raw.githubusercontent.com/Aron-he/ark-global-blacklist/main/blacklist.ini

- ghproxy代理（推荐）  
  https://ghproxy.net/https://raw.githubusercontent.com/Aron-he/ark-global-blacklist/main/blacklist.ini
